# Быстрый старт — подключите WB и Ozon к вашему ИИ

> Этот пакет даёт вашему ИИ-ассистенту (Claude, Cowork, Cursor…) прямой
> доступ к кабинетам **Wildberries** и **Ozon**: продажи, остатки, заказы,
> цены, аналитика — голосом или текстом, без открывания личного кабинета.

Выберите путь по себе. Все три приводят к одному результату.

---

## Сначала — где взять ключи (нужно один раз)

| Маркетплейс | Где | Что скопировать |
|---|---|---|
| **Wildberries** | [seller.wildberries.ru](https://seller.wildberries.ru) → Настройки → Доступ к API → создать токен (отметьте нужные категории) | один **токен** |
| **Ozon** | [seller.ozon.ru](https://seller.ozon.ru) → Настройки → API-ключи | **Client-Id** и **Api-Key** |

Ключи хранятся **только на вашем компьютере** (`~/.marketplace-mcp/cabinets.json`,
права 600). Они никогда не попадают в репозиторий, в конфиг Claude или к нам.
Можно начать с одного маркетплейса — второй добавите позже.

---

## Путь 1. Самый простой — попросите своего ИИ (без терминала)

Для тех, кто не хочет видеть командную строку.

1. Откройте **Claude Desktop** или **Cowork**.
2. Скажите ассистенту дословно:
   > «Установи мне WB и Ozon MCP — маркетплейс-коннектор.»
3. Дальше следуйте подсказкам: ассистент откроет инструкцию-навык
   (`install-skill/`, см. папку в этом пакете) и проведёт вас за руку —
   куда нажать, куда вставить ключи.

> Почему ИИ не ставит всё сам молча: песочница Cowork — это изолированный
> Linux-контейнер, она не имеет доступа к вашему компьютеру и не может печатать
> в ваш терминал (это защита). Поэтому один клик по установщику (Путь 2) всё
> равно за вами — навык просто доводит вас до него без ошибок.

---

## Путь 2. Скачать и кликнуть (рекомендуется большинству)

Не нужно ничего знать про код. Нужен только установленный **Python 3.10+**
(Windows: при установке с [python.org](https://python.org) поставьте галочку
«Add Python to PATH»).

1. Откройте страницу **Releases** на GitHub и скачайте свежий
   `marketplace-mcp-v<версия>.zip`:
   **https://github.com/ilyautov/marketplaces-mcp-ru/releases**
   <!-- TODO: создать публичный репозиторий ilyautov/marketplaces-mcp-ru -->
2. **Распакуйте** архив в удобную папку (например, в «Документы»).
3. Запустите установщик двойным кликом:
   - **macOS** — `install.command`
     (первый раз: правый клик → «Открыть», чтобы обойти Gatekeeper)
   - **Windows** — `install.bat`
   - **Linux** — в терминале `bash install.sh`
4. Установщик спросит ключи — **вставьте** токен WB и Client-Id / Api-Key Ozon
   (то, что есть; лишнее пропустите клавишей Enter).
5. **Перезапустите** Claude / Cowork. При первом запуске коннектор сам
   доустанавливает зависимости — это нормально, подождите минуту.

**Проверка** (по желанию), в терминале из папки пакета:
```bash
python serve.py ozon --selfcheck     # → "OK: ozon ready, ..."
python serve.py wb   --selfcheck
```
Или просто спросите ассистента: «Покажи мои продажи на Ozon за неделю».

## Путь 3. Технический — git / CLI

Для тех, кто живёт в терминале.

```bash
git clone https://github.com/ilyautov/marketplaces-mcp-ru
cd marketplaces-mcp-ru

# выберите свой клиент:
python3 install.py --client claude-desktop   # пишет claude_desktop_config.json
python3 install.py --client claude-code      # печатает готовые `claude mcp add`
python3 install.py --client codex            # печатает `codex mcp add`
python3 install.py --client opencode         # пишет ~/.config/opencode/opencode.json
```
<!-- TODO: создать публичный репозиторий ilyautov/marketplaces-mcp-ru -->

Ключи можно передать без интерактива:
```bash
python3 install.py --wb-token TOKEN \
                   --ozon-client-id ID --ozon-api-key KEY
```
Только посмотреть конфиг-блок, ничего не меняя:
```bash
python3 install.py --print
```

`serve.py` сам поднимает локальный venv при первом запуске — `command` может
быть любым Python 3.10+. Несколько кабинетов (магазинов): повторите с
`--cabinet shop2`, переключение из чата (`ozon_use_cabinet`). Тесты:
`python -m pytest tests/ -q` (21 офлайн-тест, ключи не нужны).
